ブランチの一覧を表示
    $ git branch -a
ローカルリポジトリにブランチを作成する。
    $ git branch <ブランチ作成元ブランチ>
    $ git checkout -b <新しいブラン名>
リモートのブランチをローカルにチェックアウト
    $ git checkout <ブランチ名>


注意：リポジトリをcloneした後、
$ git config --global user.email "yasuokatou@gmail.com"
$ git config --global user.name "YasuoKatou"
が必要とのこと
